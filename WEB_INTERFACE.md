## Client-Side Web App

BareBlocks runs entirely in the browser (no server-side processing). Serve `index.html` locally:

```bash
python -m http.server 8000
```

Open: `http://localhost:8000/index.html`

## What You'll See

1. **Client-side metadata inspector** in your browser
2. **Drag & drop** file upload
3. **Real-time metadata extraction**
4. **Formatted output** with tables and JSON

## Stop the Server

Press `Ctrl+C` in the terminal where the server is running.
